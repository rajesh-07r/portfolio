## ðŸ“¦ How to Deploy (Free)

This project is built to be deployed on **Netlify**, **Vercel**, or **GitHub Pages**.

### Option 1: Netlify (Recommended)
1.  Run `npm run build` locally.
2.  Login to [Netlify Drop](https://app.netlify.com/drop).
3.  Drag and drop the `dist/` folder into the browser.
4.  **Done!** Your site is live.

### Option 2: Vercel
1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in this directory.
3.  Follow the prompts (default settings are perfect).

### Option 3: GitHub Pages
1.  Push this code to a GitHub repository.
2.  Go to Settings > Pages.
3.  Select "GitHub Actions" or configure the build from the `main` branch.
